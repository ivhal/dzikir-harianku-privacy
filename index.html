Kebijakan Privasi

Aplikasi Dzikir Harianku · Terakhir diperbarui: 28 Juni 2026

Kebijakan Privasi ini menjelaskan bagaimana aplikasi Dzikir Harianku ("Aplikasi", "kami") menangani informasi Anda. Dengan menggunakan Aplikasi, Anda menyetujui praktik yang dijelaskan di bawah ini.

1. Data yang Kami Kumpulkan

Aplikasi ini tidak meminta, mengumpulkan, atau mengirim data pribadi (seperti nama, email, atau nomor telepon) ke server kami. Kami tidak memiliki server yang menyimpan data Anda.

Data berikut disimpan hanya di perangkat Anda dan tidak pernah dikirim ke kami:

  • Pengaturan aplikasi (jam pengingat, mode gelap)
  • Data streak dan progres dzikir harian Anda

Data ini akan terhapus jika Anda menghapus (uninstall) Aplikasi.

2. Iklan (Google AdMob)

Aplikasi menampilkan iklan melalui Google AdMob. Untuk menyajikan iklan, AdMob dapat mengumpulkan dan memproses data tertentu, termasuk identifier perangkat untuk iklan (Advertising ID) dan data penggunaan, sesuai kebijakan Google.

Selengkapnya tentang bagaimana Google menggunakan data: https://policies.google.com/technologies/partner-sites.

Anda dapat mengatur ulang atau membatasi penggunaan Advertising ID melalui pengaturan perangkat Android Anda.

3. Notifikasi

Aplikasi mengirim notifikasi pengingat dzikir pagi/petang serta pengingat doa harian sesuai aktivitas (mis. sebelum tidur, sebelum makan). Notifikasi ini dijadwalkan secara lokal di perangkat Anda, tidak melibatkan server kami, dan dapat Anda nonaktifkan kapan saja melalui menu Pengaturan di dalam Aplikasi.

4. Izin yang Digunakan

  • Notifikasi — untuk pengingat dzikir.
  • Internet — hanya untuk memuat iklan.

5. Privasi Anak

Aplikasi ini aman untuk semua umur dan tidak mengumpulkan data pribadi dari anak.

6. Perubahan Kebijakan

Kami dapat memperbarui Kebijakan Privasi ini sewaktu-waktu. Perubahan akan ditampilkan pada halaman ini dengan tanggal pembaruan yang baru.
